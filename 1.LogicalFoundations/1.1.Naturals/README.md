# Naturals

## Key binds

```txt
C-c C-l       Load
C-c C-f       Forward to next hole
C-c C-c       Cases for variable
C-c C-,       Display information about the hole
C-c C-r       Refine the hole
C-c C-space   Remove the hole
```
